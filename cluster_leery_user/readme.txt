Задачи

Разработать алгоритм выявления подозрительных пользователей

1) Загрузить данные (<дата>, <время>, <id пользователя>,<страна>, <города>, <провайдер>).

2) Преобразовать данные для выделения факторов, влияющих на подозрительность пользователя.

3) Разработать правила, по которым определять подозрительных пользователей (стат. методы или экспертное мнение).

4) Разработать модель машинного обучения для выявлениякластера подозрительных пользователей.
